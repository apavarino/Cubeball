# Installation

## Install The Plugin

1. Download the latest CubeBall JAR.
2. Stop the Minecraft server.
3. Put the JAR into the server `plugins/` directory.
4. Start the server.
5. Wait for CubeBall to generate its default `config.yml`.

## First Startup Check

Expected log output:

```text
[CubeBall] Enabling CubeBall vX.Y.Z
```

If the plugin loads correctly, you can reload its settings later with:

```text
/cb reload
```

## Recommended Server Software

- Paper for the best runtime behavior
- Spigot for broad compatibility
- Other Bukkit-compatible forks may work if they preserve falling block behavior
