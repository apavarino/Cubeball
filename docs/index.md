## What CubeBall Provides

- A physical ball represented by a configurable falling block
- Match creation directly in game from arena marker blocks
- Team management for blue, red, and spectator roles
- Automatic countdowns, round restarts, score handling, and overtime
- Configurable particles, sounds, title timings, bounce tuning, and goal animations
