# Download

CubeBall is distributed as a Bukkit plugin JAR.

## Official Pages

- [CurseForge](https://www.curseforge.com/minecraft/bukkit-plugins/cubeball)
- [Bukkit Dev](https://dev.bukkit.org/projects/cubeball)
- [GitHub Releases](https://github.com/apavarino/Cubeball/releases)

## Requirements

- Minecraft Java Edition server
- Bukkit-compatible software such as Paper or Spigot
- Java 17 for current builds

## Build From Source

```bash
./gradlew shadowJar
```

The built plugin JAR is generated under `build/libs/`.
